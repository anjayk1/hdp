Directions and How It Works

The purpose of this web application is finding the percentage of chance you have heart disease. With the use of this dataset, this website analyzes the factors of heart disease such as age, sex, chest pain type, resting blood pressure, and total cholesterol level that has been achieved. Logistic regression is used to calculate the effects of age and sex. Linear regression is used to calculate the effects of cholesterol levels and blood pressure. Decision trees are used to decide if you do have heart disease by analyzing all five factors. Then, using your chest pain level, cholesterol level, and blood pressure, the type of heart diseases you may have are given to you.

How it works is you (the user) have to input the corresponding values and select a checklist of possible symptoms. Then the data gets sent to the Python backend of the project and analyzes these factors with the data set using decision trees, logistic regression, and linear (multi and singular) regression to analyze the data. Then the backend part of the project sends back the liklihood that you have heart disease in percent form. The web application will also give a possibility of what type of heart disease it could be.

REMEMBER THAT THIS PREDICTOR IS NOT FULLY ACCURATE AND IS JUST A PREDICTION. A DOCTOR'S INSIGHT MIGHT CONTRADICT WITH THE RESULTS OF THE PROGRAM. A DOCTER'S INSIGHT WILL ALWAYS BE CORRECT.
